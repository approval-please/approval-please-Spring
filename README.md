## 📍 기술 스택
**Framework**
- Java 11
- Spring Boot 2.7.7

**Dependencies**
- Spring Security + JWT
- Spring Data Jpa
- QueryDsl
- JUnit 4

**Infra**
- Github Actions
- AWS EC2
- AWS S3
- AWS CodeDeploy
- AWS RDS
- Nginx

**Database**
- MySQL (Local, Deploy DB)
- H2 (In-memory Test DB)
